# ARcode for unity
## 要求  
###硬件  
•一個ARCORE支持Android設備或仿真器  
•用於將設備連接到開發機器的 USB 電纜  
  
###軟件  
•統一2019.4.3f1或更高版本與Android組建支持  
•ARCORE XR插件(https://docs.unity3d.com/Packages/com.unity.xr.arcore@4.2/manual/)  
   
## Android Setting  
### Package Manager  
•AR Foundation & ARCode 使用相同版本APK(兼容性問題)
